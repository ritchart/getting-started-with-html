HTML: HyperText Markup Language
-glues everything together on the WWW
-connects through hyperlinks


The Web:
-invented in 1990's
-billions of pages


Major Pieces of the web:
your computer + browser <-> the internet <-> (HTTP protocol) <-> servers


HTML is made up of:
-text content (what you see)
-markup (what it looks like)
-references to other docs (e.g., videos)
-links to other pages


HTML markup is made of tags:
-an element is both the opening and closing tags and everything between them
'b' bold tag
'em' italics tag
-you can have tags within tags


HTML attributes:
'TAG ATTR="value"'contents'/TAG'
-tags can have multiple attributes
-Anchor 'a' tag: 'a href="www.reddit.com"'derp'/a'
-Image 'img' tag: 'img src="url" alt="text"' (alt = alternate, alt is required, used for broken requests and blind people)
-IMAGE TAGS DON'T HAVE CLOSING TAGS! --> void tag
-images appear inline with text


Whitespace:
'br' tag
-'break'
-void tag
'p' tag
-'paragraph'
-not a void tag
-line break at end of p


Inline/block:
'br' is inline
-just ends a line
'p' is a block
-makes an invisible box
-it can have height and width


span/div:
-span is inline
-div is block
-these elements just contain their content
-you can attach styles/classes to them
-div causes a line break


HTML documents:
'!DOCTYPE HTML' - doctype
'html' - surrounds entire document
'head' - metadata (CSS and JS references)
'title' - inside 'head', title of page
'body' - actual content, mostly focus on this
